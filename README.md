# 웹 서비스 프로젝트

간단한 커뮤니티 기능을 포함한 웹 서비스 프로젝트입니다.

## 주요 기능

### 사용자 기능
- 회원가입
- 로그인
- 게시글 작성
- 게시글 목록 조회
- 게시글 상세 조회
- 댓글 작성

### 관리자/운영
- 관리자 기능(추가 예정)
- 모니터링 및 배포 자동화

## 기술 스택

### Frontend
- Next.js 14
- TypeScript
- Tailwind CSS

### Backend
- Node.js 20
- NestJS
- REST API

### Database
- PostgreSQL
- Redis (캐시)

### Infra / DevOps
- Docker
- Nginx
- GitHub Actions (CI/CD)
- AWS (EC2, RDS, S3)

### Testing / Quality
- Jest
- Supertest
- ESLint
- Prettier
